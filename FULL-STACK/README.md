# Retailintel (RetailIntern)

Retailintel is a full-stack web application focused on retail store management & analytics.

## Features

- **Real-time Inventory**: Track stock levels, product details, and movements in real-time.
- **Customer Profiles & History**: Manage customer data and view purchase history.
- **Sales Analytics & Charts**: Visualize sales performance and trends.
- **ML Demand Predictions**: Predict future product demand using machine learning.

## Tech Stack

- **Framework**: Next.js 15
- **UI Library**: React 19
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
